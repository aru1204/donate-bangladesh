Important FAQ

What is Documont Object Model (DOM)?

The DOM is a programming interface for web documents that allows JavaScript to manipulate HTML elements.

How do you select an element from DOM?

To select an element from the DOM in JavaScript you use -

  • ID
  • Class
  • Tag Name
  • CSS Selector
What is event delegation in the context of the DOM, and why is it useful?

Event delegation involves attaching a single event listener to a parent element to manage events for its children. It's useful because it improves performance, simplifies code, and handles dynamically added elements.

What is the purpose of querySelector() in the DOM?
